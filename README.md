[fastapi-naive-admin](https://github.com/cary997/fastcow-devops)

### 本地开发环境

- node.js v20.11.1
- npm 10.4.0

### 开发环境安装

#### 安装依赖

```shell
npm i
```

#### 启动

```shell
npm run dev
```

#### 打包

```shell
npm run build
```
